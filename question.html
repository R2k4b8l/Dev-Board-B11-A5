Question-2: What is the difference between innerHTML, innerText, and textContent ?
Answer:InnerHTML returns all HTML structure inside the elements and it ignore the hidden element.InnerText returns only text .It's not interpret the HTML tags and it is one of the difference of innerHTML and innerText,innerText ignores the hidden element.textContent returns all text including hidden element and does not parses HTML tags.
Question-3: What is event delegation in the DOM?
Answer: Instead of adding an event listener to each child element, we add one listener to the parent and let it handle events for all its children.Here we use only one event listener.Event delegation use when we handling clicks on lists, tables, or menus and here elements are added dynamically.
Question-4: What is event bubbling in the DOM?
Answer: Event bubbling is when an event starts at the target element and then moves up (bubbles) through its parent elements until it reaches the document.We use event.stopPropagation() to stop event bubbling.
Question-5: How do you create, add, and remove elements using JavaScript?
Answer: To create an element we use .createElement. Newly created element can be modified using various command such as .innerHTML and so on. To add a new element we can use either .appendchild or .append. Lastly, to remove an element we use .removeChild.
